About GreenGuard

GreenGuard is a simple open-source web project that helps users understand eco-friendly habits and encourages pollution reduction through small daily actions.

Eco Score Checker

Enter how many eco-friendly actions you did today.

Key Features

Eco Awareness

Educates users about daily green habits.

Simple Score

Gives a basic eco score based on user actions.

Open Source

Allows contributors to improve features and content.

Eco Tips

  • Use public transport or bicycles.
  • Switch off lights and fans when not in use.
  • Reduce plastic usage.
  • Plant trees and protect green spaces.
